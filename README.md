# Final Frontier

An interactive map of our solar system, built in Godot 4 with help from Claude. Fully explorable and with its own internal wiki, providing context and information about each celestial body. This project is a proof of concept for a larger space exploration game, called Final Frontier. The basic vision is "Sid Meiers Pirates in Space".

![img](docs/img/planet_view.png)

## Status

This repository is a **completed proof of concept** (tagged `v1.0-poc`). It is no longer being actively developed. The follow-up project rebuilds the simulation in 3D from the ground up; modules from `core/`, the almanac, and the UI panels are intended for reuse. See [`docs/HANDOFF.md`](docs/HANDOFF.md) for what carries over and what does not.
