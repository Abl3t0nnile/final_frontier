# Planet Viewer - Godot 4 Plugin

## Overview

This plugin provides a planet viewer built with Godot 4. It allows users to view and interact with planets in a 2D environment. It is intended to be used in the Final Frontier project. The scope of this tool is to render a planet either procedurally or from a texture and let users rotate the planet in two dimensions. There should also be a way to provide markers for points of interest. These markers should be able to display a name and be to be clicked and send a signal.

## Target Look

1-Bit styling with outline. Planet texture should have 6 brightness levels to create the illusion of depth. The texture is mapped to a sphere and lit using floyd-steinberg dithering.
